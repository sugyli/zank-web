# zank - 获取验证码

## 接口
```
/captcha/phone/get/register
```

## 请求体
| name     | type     | desc     |
|----------|:--------:|:--------:|
| phone    | strinf   | 获取验证码的手机号码 |
| test     | int      | 固定址1，用于开发使用，（目前用于界面提示）|

## 返回体
```json5
{
  "status": true, // 是否成功
  "message": "获取验证码成功！", // 消息
  "data": 7387 // 验证码
}
```
